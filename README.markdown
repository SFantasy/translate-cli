Translate-cli
---

Translate between Chinese and English.

- Use the api of [fanyi.baidu.com](fanyi.baidu.com)

- Need [jsawk](https://github.com/micha/jsawk)

### NOTE

#### Installation

    git clone https://github.com/SFantasy/translate-cli.git
	cd translate-cli
	chmod 775 dic.sh
	ln dic.sh /bin/dic
	
#### Usage

    $ dic 你好
	Hello
	$ dic 火车
	Train
