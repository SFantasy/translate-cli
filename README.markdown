Translate-cli
---

Translate between Chinese and English.

- Use the api of [fanyi.baidu.com](http://fanyi.baidu.com)

- Dependence: [JSON.sh](https://github.com/dominictarr/JSON.sh)

### NOTE

#### Installation

    git clone https://github.com/SFantasy/translate-cli.git
	cd translate-cli
	chmod 775 dic.sh
	ln dic.sh /bin/dic
	
#### Usage

    $ dic 你好
	Hello
	$ dic 火车
	Train
	$ dic Today
	今天
